## 📋 Quick Reference

**Setting Era:** ~7200 CE (general), 7236-7243 "When Neptune Wakes", 7251+ "Hollow Orbit"

**Major Factions:**
- Earth (peaceful, artistic, unified)
- Mars (industrial, capitalist, pragmatic)
- Titan (luxurious, biotech, hedonistic)
- Neptune (militarized, religious, expansionist)
- Plus: Mercury, Venus, Luna, LaGrange, Jupiter, Saturn, Uranus, Pluto

**Key Concepts:**
- Myce-Steel™ - Self-healing fungal alloy
- ELM-00 - Revolutionary mech prototype
- AELITH - Bio-robot AI hybrid pilot
- NNC - New Neptune Coalition (rebels become tyrants)
- Phosphora - Sentient alien planet actively resisting colonization
