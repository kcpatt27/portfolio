Project Overview

NextDish is an AI-powered recipe platform born from a baking session—the idea that started my dev journey. Using ChatGPT to brainstorm flavor combinations while baking cakes led to the realization that this process could be systematized.

Core Features

  • AI-powered recipe generation from text or images
  • Social features for recipe sharing
  • Advanced search and filtering
  • User-generated content platform

Technology Stack

  • Frontend: Next.js, React
  • Backend: Supabase
  • AI: OpenRouter API
  • Database: PostgreSQL

Technical Specifications

Detailed technical requirements and implementation details.

API Endpoints

/api/recipes/generate - POST
/api/recipes/search - GET
/api/users/profile - GET
/api/social/share - POST

Data Models

  • Recipe: title, ingredients[], instructions[], metadata
  • User: profile, preferences, social connections
  • AI Request: prompt, response, model used

System Architecture

High-level system design and component relationships.

Frontend Architecture

  • Next.js App Router for routing
  • React components with TypeScript
  • Tailwind CSS for styling
  • React Query for data fetching

Backend Architecture

  • Supabase for database and auth
  • Edge functions for API logic
  • OpenRouter for AI model access
  • PostgreSQL for data storage
